Програмуємо для Android

Гурток Миколаївської гімназії №3

Сучасно

Користуєшся ОС Android та вважаєш, що в Google Play не вистачає саме твого додатка?

Легко

Тобі 11-13 років і ти тільки починаєш програмувати але вмієш складати пазли?

Цікаво

Використовуємо хмарне середовище від Массатчусетського технологічного інституту

Серйозно

Що далі? Переходь до вивчення мови програмування з App Inventor Java Bridge Code Generator

З "MIT App Inventor 2 (AI2)", Ви зможете:
реалізовувати власні ідеї додатків для Android
розробляти прості, але цікаві мобільні ігри з AI2
використовувати AI2 для керування платою Arduino
створювати додатки для послуг, пов'язаних із IOT (Internet of things)

MIT App Inventor - це середовище програмування, розроблене вченими з Массачусетського Технологічного Інституту (MIT), призначене для розробки додатків для мобільних пристроїв (смартфонів і планшетних комп'ютерів), що працюють на операційній системі Android.

Мова цього середовища дуже проста, оскільки з самого початку створювалась для школярів. Для того, щоб програмувати в ньому, не потрібно писати рядки коду, як це відбувається у "дорослих" мовах програмування. Для створення програми в MIT App Inventor ми будемо просто перетягувати блоки, збираючи їх в програму, як пазли.

Ми розглянемо поняття об'єкта та його властивоті. З'ясуємо правила та засоби впливу на ці властивості. Вивчимо події, пов'язані з компонентами програми та навчимося програмувати поведінку об'єкта.

Наші додатки будуть використовувати сенсори, камеру, мікрофон, динаміки, годинник, декілька екранів та інші вбудовані можливості смартфонів. Навчимося використовувати змінні для "спілкування" між датчиками та додатком.

Розробляючи інтерфейс додатка, нам доведеться розібратися з поняттями UI та UX, позиціонуванням компонентів (layout), ознайомитися з кольоровими схемами та користуватися системою координат.

Всі роботи ми будемо зберігати у власному портфоліо. Для цього доведеться начитися працювати з Google sites.

Ми маємо дізнатися про методи генерування ідей щоб стати винахідниками. Навчитися коротко та зрозуміло формулювати думки для написання інструкцій з використання додатків й створення їх опису для GooglePlay.

  1. Щоб потрапити у середовище програмування переходимо за посиланням ai2.appinventor.mit.edu
  2. Для реєстрації використовуємо свій Google-аккаунт.
  3. Для тестування створеного додатка нам знадобиться MIT AI2 Companion. Його можна встановити на смартфон або використовувати в режимі емуляції.
  4. Всі інструкції, текстові та відеоматеріали, вправи та додаткові файли можна знайти на Дистанційному курсі
